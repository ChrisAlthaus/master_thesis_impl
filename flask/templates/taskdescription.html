Information & Task Desription

The following user study investigates the retrieval accuracy of an artistic search engine.
This search engine should allow historians to find visual links between artworks.

Through history, many artists draw inspiration from each other by using similar painting styles, themes and motifs. 
Or painters copied the scene structure.
This search engine tries to retrieve images which are similar to the query images, taking into account two aspects:

  • Arrangement of Human Poses: The posture of characters in the query and retrieved image should be similar.
    Sometimes painters copied a theme by adopting the configuration of human poses of another artwork.
    Or they adoped just one small gesture like the arm posture, e.g. Michelangelo's David or the hand-on-the-hip posture
    which was often used during Renaissance.
  • Arrangement of Objects in the Scene: The scene composition in the query and retrieved image should be similar.
    This means that there are multiple objects which are contained in both images,
    e.g. mountain, trees, buildings or chairs.

This user study seperates the retrieval of paintings w.r. to similar human poses and the retrieval of similar scenes.
Therefore, the first subset of the following pages shows retrieval results w.r. to similar human poses and the second subset
of the following pages shows retrieval results w.r. to similar objects in the images. There will be a reminder page
when the task is changed to scene/object similarity.

Notes on relevance selection:

  • Human Poses:
    • For 1 person in the query image: If there are multiple poses in the retrieved image and one matched the query pose,
      the image also should be relevant.
      In general, if the query pose is in a retrieved image, this image should be relevant.
    • For >1 persons in the query image: The prime poses (not too small and well visible) of the query image
      should be present in the retrieved image.
  • Object in Scenes:
    • The retrieved image may contain a subset of the query image's objects.
      Not all objects necessarily have to be present in the retrieved image (e.g. like the house in the mountain painting above).
The next pages show retrieval results which are ranked from 1 to 20. The query image is presented in the left upper corner.
Please select all images which you consider relevant for the query image under consideration of the task description
at the top of each page.
