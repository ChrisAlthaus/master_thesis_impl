Information & Task Desription

The following user study investigates the retrieval accuracy of an artistic search engine.
This search engine should allow historians to find visual links between artworks.

Through history, many artists draw inspiration from each other by using similar painting styles, themes and motifs. 
Or painters copied the scene structure.
This search engine tries to retrieve images which are similar to the query images, taking into account two aspects:

  • Arrangement of Human Poses: The posture of characters in the query and retrieved image should be similar.
    Sometimes painters copied a theme by adopting the configuration of human poses of another artwork.
    Or they adoped just one small gesture like the arm posture, e.g. Michelangelo's David or the hand-on-the-hip posture
    which was often used during Renaissance.
  • Arrangement of Objects in the Scene: The scene composition in the query and retrieved image should be similar. This means that there
    are multiple objects which are contained in both images, e.g. mountain, trees, buildings or chairs.

The next pages show retrieval results which are ranked from 0 to 50. The query image is presented in the left upper corner.
Please select all images which you consider relevant for the query image. The search engine considers the arrangement of similar persons more important than
the arrangement of similar objects. Therefore please primarily focus on relevance based on similar human poses.

Note:

  • Please keep the browser open for the whole session, otherwise data may be lost.
  • You can also select nothing, if no image is relevant.
  • When finished the user study please close the browser window.
  • There are 40 pages in total, followed by a logout screen.
  • In case there's an error please restart the browser.
